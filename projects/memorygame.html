Scratch Memory Game

Here, I created a Scratch Memory Game by following the tutorial of ShiftClickLearn's YouTube video. By creating different kinds of effects and animations, I created a card game to match cats. But some cats had unique abilities, such as giving you a power up or an extra heart, or even getting rid of a heart. I created the code by following the tutorial which basically creates different kinds of variables for each 11 card and each action that the user does. For example, selecting a card is CardSelect1, etc. By defining them as variable, we are able to know when to continue doing another action. These actions are usually defined as well in a function that we write on our own.
